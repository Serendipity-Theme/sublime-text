# Serendipity for Sublime Text

Elegant, minimal, and clean color palette for your tools.

See other interfaces at the [official website](https://www.michaelandreuzza.com/vscode/serendipity/).

## Available themes

- **Midnight** — dark
- **Morning** — light
- **Sunset** — dark

## Installation

Copy `.tmTheme` files to `Packages/User/` or install via Package Control. Set **Preferences → Color Scheme** to Serendipity Midnight / Morning / Sunset.

Available files: `serendipity-midnight.tmTheme`, `serendipity-morning.tmTheme`, `serendipity-sunset.tmTheme`.
